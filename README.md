## Getting Started
First, install dependencies:

```bash
npm install
# or
pnpm install
```

Second, fill out sample .env file & push to prisma:
```bash
npx prisma db push
```

Last, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
